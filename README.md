This is a system that utilizes Machine Learning to analyse mobile money transations
and unsusual transations which may indicate fraud.
The system uses the Isolation Forest Algorithm to detect anomalies from usual transaction behavior/patterns
based on the transaction amount, transation time and transation frequency.

What I learnt in this project:

Python – Programming language
Pandas – Data manipulation and analysis
Scikit-learn (sklearn) – Machine learning library
Isolation Forest – Anomaly detection algorithm
Matplotlib – Data visualization and scatter plots
